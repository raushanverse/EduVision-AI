# 📖 Overview

EduVision AI is an intelligent career guidance platform that helps students identify the most suitable career path using Machine Learning.

The application analyzes academic performance, interests and goals to generate personalized recommendations and learning roadmaps.

Designed with a modern interface and AI-powered prediction engine, EduVision AI makes career planning simple, interactive and data-driven.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/raushanverse/EduVision-AI.git
```

---

## Change Directory

```bash
cd EduVision-AI
```

---

## Create Virtual Environment

```bash
python -m venv venv
```

---

## Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux

```bash
source venv/bin/activate
```

---

## Install Requirements

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
streamlit run app.py
```
